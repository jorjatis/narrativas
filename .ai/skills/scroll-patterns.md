# Scroll & Animation Patterns

## 1. Reveal básico

- IntersectionObserver
- Añadir clase .is-visible
- Transición en CSS

## 2. Contador numérico

- IO como trigger
- requestAnimationFrame
- No setInterval

## 3. Sticky progresivo

- position: sticky
- Cálculo puntual
- Throttle si necesario

## 4. Horizontal section

- Scroll-snap si es simple
- GSAP solo si hay scrub real

---

Siempre priorizar:

CSS > IO + CSS > GSAP
